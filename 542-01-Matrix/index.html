题目描述

Given a matrix consists of 0 and 1, find the distance of the nearest 0 for each cell.

The distance between two adjacent cells is 1.

具体样例与描述见https://leetcode.com/problems/01-matrix/description/

分析

直观上我们可以使用BFS的思想，首先找到所有的值为0的点，将其Result设为0，并加入到队列Q中，并设置vis的值为true，每次取出队列中的一个点，更新其上下左右点的值，如果被更新点vis值为false，则加入队列Q，并设置vis值为true。直至Q为空。

这种算法由于一个点只会放入队列一次，所以时间复杂度是O(RC)的，但空间复杂度，由于要存储队列和vis，Result矩阵，所以空间复杂度是O(RC)的。

接下来介绍这题的最终解法，其实想到了就很简单，实现起来也很短：我们从(0,0)点，向右想下遍历，对于非0点，更新为其左/上点中值较小的点，之后再从(R-1,C-1)向左向上遍历，对于非0点，更新为其左/上/本身中的最小值。

简单证明一下其正确性；由于值为0的点其距离必为0，所以对0点这个算法是正确的，而对于非0点(i,j)，其距离等于min(他上面所有0点与其本行左侧所有0点的最小距离,他下面所有0点与本行右侧所有0点的最小距离），因而这个解法是正确的。
