强调

使用**A**表示加粗，Hello

链接与邮件地址

  1. <https://www.google.com>表示 https://www.google.com
  2. [G](https://www.google.com)表示G
  3. 使用引用[G][Google]然后声明[Google]: https://www.google.com与2结果相同
  4. 使用[G][]然后在其他位置声明[G]: https://www.google.com与3一致

数学语法

##插入
行内使用$str$,块使用$$str$$。举个例子
$$x=100*y+z-19/22+100\%2$$

##特殊符号
空格：\Space   $:\$  %:\% 
{:\lbrace  }:\rbrace 

##上下标
^表示上标，_表示下标。如果字符多于一个用{}来当成整体

##分数
两种写法\frac{分子}{分母}更小，\cfrac{分子}{分母}较大。比较见下（左边为frac右边为cfrac）
$$\frac{1}{3}与\cfrac{1}{3}$$

##开方
使用\sqrt[次数]{被开方数}表示开方。栗子见下
$$\sqrt[3]{x}$$
